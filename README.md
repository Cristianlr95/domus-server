# Domus Backend

Base backend de `Domus` usando el mismo stack principal de `Atleta`:

- Java 21
- Spring Boot
- Spring Security
- OAuth2 Client y Resource Server
- Spring Data JPA
- PostgreSQL
- Flyway
- OpenAPI
- Maven Wrapper

## Comandos

- `./mvnw spring-boot:run`
- `./mvnw test`
- `./mvnw verify`

En Windows:

- `mvnw.cmd spring-boot:run`
- `mvnw.cmd test`
- `mvnw.cmd verify`
